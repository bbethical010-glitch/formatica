# Formatica v1.0.0 - Official Release

**Formatica** is a powerful, fast, privacy-first desktop application for media conversion, compression, and extraction. All processing happens **locally on your computer** — nothing is uploaded to the cloud.

---

## 📦 What's Inside

This release contains:
- **Formatica_1.0.0_x64-setup.exe** (2.9 MB) - Windows Installer (NSIS) - **RECOMMENDED**
- **Formatica_1.0.0_x64_en-US.msi** (4.5 MB) - Windows Installer (MSI)
- **SETUP_GUIDE.md** - Complete installation and usage guide

---

## ⚡ What You Can Do

### Video Processing
- ✅ **Compress Video** - Reduce file size while maintaining quality
- ✅ **Convert Video** - MP4, MKV, MOV, AVI, WEBM, GIF
- ✅ **Extract Audio** - MP3, AAC, WAV, FLAC, OGG, M4A, OPUS
- ✅ **Download Media** - Save YouTube and online videos locally

### PDF Tools
- ✅ **Merge PDF** - Combine multiple PDFs into one document
- ✅ **Split PDF** - Break PDF by page count or custom ranges
- ✅ **Greyscale PDF** - Convert color PDFs to black & white (print cost savings)
- ✅ **Convert Document** - DOCX, PDF, XLSX to any format

### Image Processing
- ✅ **Images to PDF** - Combine JPG, PNG, WEBP, BMP into one PDF
- ✅ **Convert Image** - PNG, JPG, WEBP, GIF, BMP, TIFF with quality control

## 🚀 Quick Start

1. **Download** - Choose either installer (both are identical):
   - **Windows NSIS** (faster, smaller) - `Formatica_1.0.0_x64-setup.exe`
   - **Windows MSI** (traditional installer) - `Formatica_1.0.0_x64_en-US.msi`

2. **Install** - Double-click and follow the wizard
   - Installation takes ~2 minutes
   - Auto-downloads required tools on first launch

3. **Use** - Launch Formatica from Start Menu
   - All processing is **instant and local**
   - No cloud upload, no tracking

## 🎯 How to Install

1. Download the installer file
2. Right-click → Run as Administrator (recommended)
3. Click Next through the wizard
4. Choose installation location (default is fine)
5. Click Install and wait 1-2 minutes
6. Launch Formatica when done!

---

## ⚠️ First Run

On first launch, Formatica will:
- Download yt-dlp (~10 MB) for media downloading
- Optionally suggest installing LibreOffice for document conversion
- Show a brief onboarding tour

This setup is **one-time only** and takes about 2 minutes.

---

## 🛠️ Troubleshooting

**If installation fails:**
- Try running as Administrator
- Ensure 500MB free disk space
- Check Windows 10/11 64-bit version
- Disable antivirus temporarily during install

**If converter tools don't work:**
- Click the ⚡ "Fix Now" button in the app
- Or manually install from the links provided
- Restart Formatica after installing dependencies
