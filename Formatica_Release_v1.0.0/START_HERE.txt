✨ WHAT YOU CAN DO WITH FORMATICA:

Video Tools:
  • Compress Video - Reduce file size with custom quality
  • Convert Video - MP4, MKV, MOV, AVI, WEBM, GIF
  • Extract Audio - MP3, AAC, WAV, FLAC, OGG, M4A, OPUS
  • Download Media - Save YouTube videos locally

PDF Tools:
  • Merge PDF - Combine multiple PDFs
  • Split PDF - Break PDFs into smaller files
  • Convert Document - DOCX, PDF, XLSX, ODT, PPTX, TXT...
  • Greyscale PDF - Convert color PDFs to B&W

Image Tools:
  • Images to PDF - Combine JPG, PNG, WEBP, BMP into PDF
  • Convert Image - JPG, PNG, WEBP, GIF, BMP, TIFF

❓ WHICH INSTALLER SHOULD I USE?

NSIS (Formatica_1.0.0_x64-setup.exe) - RECOMMENDED
  • Smaller file (2.9 MB)
  • Faster to download and install
  • Standard Windows installer
  • Works perfectly on Windows 10/11

MSI (Formatica_1.0.0_x64_en-US.msi) - ALTERNATIVE
  • Enterprise-compatible
  • Windows Update friendly
  • Larger file (4.5 MB)
  • Choose if you prefer MSI format

Both are identical in functionality. Pick either one!
