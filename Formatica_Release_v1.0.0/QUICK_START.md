# 🚀 Formatica - 30-Second Quick Start

## Download & Install

1. **Download**: `Formatica_1.0.0_x64-setup.exe` (2.9 MB)
2. **Run**: Double-click → Follow wizard → Click Install
3. **Wait**: 2 minutes for installation
4. **Launch**: Click "Start Formatica" or find in Start Menu

## First Run

Formatica will automatically:
- ✅ Download yt-dlp (~10 MB) for media downloading
- ✅ Ask about LibreOffice (skip if not needed for docs)
- ✅ Show feature overview

Takes ~2 minutes total.

## Start Using

1. Click any feature tile on the home screen
2. Drag & drop your file OR click to browse
3. Choose output folder
4. Click Convert/Compress/Extract
5. Done! File saved to your chosen folder

## Example Workflows

### Convert a Video to MP4
1. Click "Convert Video"
2. Drop your file (MKV, MOV, AVI, WEBM)
3. Choose "mp4" format
4. Pick save location
5. Click Convert → Done in seconds

### Compress Video & Reduce Size
1. Click "Compress Video"
2. Drop your MP4 or MKV file
3. Adjust quality slider (lower = smaller file)
4. Choose save location
5. Click Compress → Dramatically smaller file 🎉

### Extract Audio from Video
1. Click "Extract Audio"
2. Drop your video file
3. Choose audio format (MP3, WAV, AAC, etc.)
4. Pick save location
5. Click Extract → Perfect audio file 🎵

### Merge Multiple PDFs
1. Click "Merge PDF"
2. Drop all PDF files you want to combine
3. Files combine in order you selected
4. Choose save location
5. Click Merge → Single PDF with all pages 📄

### Combine Images into PDF
1. Click "Images to PDF"
2. Drop JPG, PNG, WEBP images
3. They'll combine in order
4. Choose save location
5. Click Combine → Single PDF document 📸

## Tips & Tricks

- **Faster with Hotkeys**: Right-click on files → "Open with Formatica"
- **Keyboard Shortcuts**: Type first letter of feature to quickly access it
- **Theme**: Toggle dark/light theme in top-right
- **Your History**: All conversions logged in Recent Activity panel
- **Quality vs Speed**: Higher quality = slower, larger files
- **Resolution**: Lower resolution = faster encoding + smaller files

## Troubleshooting (30 seconds)

**"FFmpeg not found"**
- Click ⚡ Fix Now button → Auto-installs it

**"LibreOffice not found" (for docs)**
- Click ⚡ Fix Now → Download link provided

**"Conversion is slow"**
- That's normal! Encoding takes time
- App isn't frozen, it's just processing
- Lower quality/resolution speeds it up

**App won't start**
- Restart Windows
- Run installer again
- Check you have 500MB free space
