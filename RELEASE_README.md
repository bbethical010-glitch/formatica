# Formatica v1.0.0 - Installation & Release Guide

## 📦 What's Included

Formatica is an all-in-one media conversion and document processing desktop application. Ready to download and install on Windows systems.

### System Requirements
- **OS:** Windows 10 or later (64-bit)
- **RAM:** 4GB minimum (8GB recommended)
- **Storage:** 500MB free space for installation + dependencies
- **Internet:** Required for first-run setup (downloads ffmpeg and yt-dlp)

## 🚀 Installation Steps

### For End Users - Easy Install Method:

1. **Download the installer**
   - Get: `Formatica_1.0.0_x64-setup.exe` (2.9 MB)
   - Save anywhere on your computer

2. **Run the installer**
   - Double-click `Formatica_1.0.0_x64-setup.exe`
   - Windows may ask for permission - click "Yes" to allow
   - Follow the on-screen prompts

3. **First Run - Automatic Setup**
   - App will automatically download required tools:
     - **ffmpeg** - Video/audio processing engine
     - **yt-dlp** - Media downloader
   - LibreOffice (300MB) may need manual install for document conversion
     - App will prompt you with a download link if needed
   - Keep the app open during this setup (~2-5 minutes)

4. **Start Using**
   - Formatica appears in your Start Menu
   - Click to launch and begin converting, compressing, and processing media

## 🛠️ Troubleshooting

### "LibreOffice not found"
- App needs LibreOffice for document conversion
- Click the "📥 Download LibreOffice" button when prompted
- It's free and the app will guide you through installation
- Restart Formatica after installing LibreOffice

### "ffmpeg or yt-dlp missing"
- These are downloaded automatically on first run
- If missing after setup, click "⚡ Fix Now" in the app
- Check your internet connection during initial setup

### Video conversion is slow
- Works fine on normal speeds - varies by file size and CPU
- GPU acceleration is enabled if available (NVIDIA/AMD recommended)
- Close other applications to free up resources

### App won't start
- Try restarting your computer
- Reinstall by running the setup.exe again
- Ensure Windows is fully updated

## 📖 Usage Examples

### Convert Word Document to PDF
1. Open Formatica → Click "Convert Document"
2. Drag your .docx file or click "Browse"
3. Choose output format (PDF, .docx, .txt, etc.)
4. Click "Browse" to select save location
5. Click "Convert Document" and wait for completion
6. Click "Open folder" to see your result

### Compress Large Video
1. Formatica → "Compress Video"
2. Add your video file(s)
3. Set resolution (1080p, 720p, etc.)
4. Adjust Quality slider (higher = bigger file, better quality)
5. Choose speed preset (fast = quicker, uses more CPU)
6. Select save location
7. Click the compress button

### Download YouTube Video
1. Formatica → "Download Media"
2. Paste the YouTube URL
3. Choose save location
4. Click "Start Download"
5. Video saves as MP4
